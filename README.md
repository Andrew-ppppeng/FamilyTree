一个简单的族谱生成工具
---

### Demo地址：
https://andrew-ppppeng.github.io/familyTree/demo/FamilyTree.html


### 功能：

1. 动态生成族谱布局
2. 给定任一家庭成员，计算所有其他家庭成员的称呼


### Quickstart

git clone, 并将data.js替换为你家族的信息
